Guide: help on how to use the website

Chart

  • Select/deselect series by clicking on the corresponding one
  • Click & drag over the chart to zoom into a section

Table

  • Click column headers to reorder data in ascending/descending orders
  • Show more/less entries using the dropdown in the upper-left corner
  • Search for data in upper-right text box

Dictionary

  • Ret - Retirements
  • NC - Not Classified
  • DNQ - Did Not Qualify
  • DNPQ - Did Not Pre Qualify
  • DSQ - Disqualified
  • DNS - Did Not Start
  • DNP - Did Not Practice
  • Ex - Excluded
  • WD - Withdrawn
