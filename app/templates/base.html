Welcome to Formula 1 Crash Stats

This website parses the HTML source code of each Formula 1 season's Wikipedia page to gather data about DNFs throughout the seasons.

Please note that this website is still being developed. I am still working on making the parser more reliable and organizing the data in a more meaningful charts before working on the website's design. You can follow the progress here https://github.com/Adamouization/Formula1-Crash-Stats
