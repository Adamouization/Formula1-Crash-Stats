Formula 1 DNF Stats

This website parses the HTML source code of each Wikipedia page for every Formula 1 season in order to gather data about the DNFs that occurred throughout the seasons.

Please note that this website is still being developed as I am working on collecting more data through the parser and exploring new methods to display it in more meaningful charts and tables. For a better experience, please visit the website on a desktop device rather than a mobile device.

If you have any suggestions or critiques, please post them here.

Guide to use the website

Chart

  • Select/deselect series by clicking on the corresponding one
  • Click & drag over the chart to zoom into a section

Table

  • Click column headers to reorder data in ascending/descending orders
  • Show more/less entries using the dropdown in the upper-left corner
  • Search for data in upper-right text box

Dictionary

  • Ret - Retirements
  • NC - Not Classified
  • DNQ - Did Not Qualify
  • DNPQ - Did Not Pre Qualify
  • DSQ - Disqualified
  • DNS - Did Not Start
  • DNP - Did Not Practice
  • Ex - Excluded
  • WD - Withdrawn
